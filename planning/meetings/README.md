# Team Meetings

Store one Markdown file per meeting using the format `YYYY-MM-DD-topic.md`.

Required sections:

- Date and time (ICT, UTC+7)
- Attendees
- Agenda
- Decisions
- Action items with owner and deadline
- Blockers
- Links to related Jira work items and pull requests

Do not use meeting notes as a replacement for Jira. Decisions and context belong here; actionable work belongs in Jira.
