# Demo Recording Checklist

- [ ] Release candidate commit is checked out.
- [ ] Fixed Ollama models are present.
- [ ] Index manifest matches the data release.
- [ ] Demo input and expected behavior were reviewed by QA.
- [ ] Streamlit starts from the repository root.
- [ ] Browser zoom and font size are readable.
- [ ] No API keys, personal paths, tokens, or unrelated tabs are visible.
- [ ] Final JSON is shown in full.
- [ ] Results are not edited after inference.
- [ ] Audio and screen recording are checked before the full take.
- [ ] Final video is stored under `videos/final/` and linked in the release checklist.
