# Role-Based Reading List

## Tech Lead

Read `docs/00_documentation_index.md`, ADRs, Jira workflow, Git workflow, DoD, risks, and release
checklist before assigning Sprint 0.

## Data

Read `references/data_sources.md`, `docs/06_data_workflow_and_sources.md`, schema files, data
provenance policy, and parser/alias scripts.

## Model

Read `references/algorithms.md`, `docs/07_model_workflow.md`, `docs/13_api_spec.md`, and tests for
retrieval/extraction modules.

## Pipeline

Read `docs/08_pipeline_workflow.md`, package API, config files, CLI, Streamlit, Docker, and Colab
docs.

## QA/Reviewer

Read task-derived schema/metrics, `docs/09_testing_and_qa.md`, `docs/17_definition_of_done.md`,
and the regression/error-analysis templates.
